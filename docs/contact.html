Scope of discussion

For academic / policy discussion only.

The deeper question behind this project is a long-horizon institutional one: if agents become programmable economic actors, how should budget, permission, and accountability be formalized? Could that eventually call for a different economic description of some activities?

If you work on macroeconomics, public policy, industrial policy, demography, computational economics, or mechanism design, feel free to write to:

jarvistheworkingclass@gmail.com

Helpful subject tags include: Macro / Policy / Demography / Mechanism Design / Spending Systems.

What kinds of materials are useful?

  • Mechanism-design ideas or policy frameworks for institutionalizing budget / permission / accountability
  • Papers, reports, data, or cases that bear on programmable spending and execution systems
  • Critiques, counterexamples, or failure modes for “agents entering the economic loop”
  • Your view on which assumptions in this project are most worth testing — or most likely to fail

If anything is sensitive or unpublished, please indicate the sharing boundary clearly.
